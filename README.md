# Setup

## setup venv enviroment 

for mac: 
```bash
source venv/bin/activate
```

for windows: 
```bash
./venv/Scripts/activate
```

## install requirements

```bash
pip install -r requirements.txt
```

# How to run

```bash
python main.py
```

# Testing

in root directory run:

```bash
pytest
```

for debugging:

```bash
pytest -s
```
